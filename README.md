# 2. 게임 소개
* 게임 이름 : 던전 식당
* 게임 컨셉

    * 던전에서 식당을 운영하며 모험가에게 요리를 제공하는 독특한 게임 배경
  
    * 게임의 주 목표는 모험가를 고용해 파티를 꾸려 다양한 테마의 던전을 돌파하여 획득한 재료를 활용해 요리 레시피를 개발하는 것

    * 새로운 던전, 요리, 캐릭터등을 추가함으로써 업데이트를 통해 계속해서 확장할 수 있는 가능성
